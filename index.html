Alarm Clock

We created an Internet of Things alarm clock that opens your window blinds and eases your waking up in the morning. The clock is set in the much like a parking dial. The next day it will signal for your blinds to open 1 hour before ringing to give a more natural wake up process.

Dancing Queez

A small mobile web-app music quiz that allows the user to generate a music quiz using their own Spotify-playlists or username. The project utilized AngularJS and Node.js to call information from Spotify's Web API.

Balls To The Walls

A fun 2-person game of virtual squash played in a 3D stereoscopic environment using the Microsoft Kinect 2.0. Incorporates Microsoft Kinect 2.0 and a 4x3 meter 4k in Unity 3D.
